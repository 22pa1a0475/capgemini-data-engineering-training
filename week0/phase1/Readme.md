# Objective

In this phase, the goal is to build a strong foundation in PySpark by understanding basic DataFrame operations and how they relate to SQL queries.

# Problem Summary

We worked with a simple customer dataset and performed basic operations to understand how data is handled in PySpark.
The tasks included:
1. Displaying data
2. Filtering records
3. Selecting specific columns
4. Performing basic aggregation

# Approach
Created a sample dataset using PySpark
Loaded and viewed the data using show()
Applied basic transformations:
o Filtered customers based on city and age
o Selected required columns like name and city
Performed aggregation:
o Counted number of customers in each city
Compared SQL queries with PySpark implementation
# Key Transformations Used

1.show() → to display dataset
2.filter() → to apply conditions
3.select() → to retrieve specific columns
4.groupBy() → to group data
5.count() → to calculate totals
