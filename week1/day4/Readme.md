Week 1 Day 4 – Date Functions, Null Functions, String and Regex Processing

1. Objective
   In this session, the objective is to understand and apply different types of SQL functions including date functions, null handling functions, string functions, regex functions, and mixed data transformations to solve real world data problems.

2. Problem Summary
   This day consists of multiple practice modules:

Date Functions Practice:
Working with date and timestamp data
Performing date extraction, formatting, and calculations 

Null Functions Practice:
Handling missing values using different null handling techniques
Applying functions like ISNULL, COALESCE, and NULLIF 

Regex Functions Practice:
Extracting patterns from text data
Working with emails, phone numbers, and mixed values using regular expressions 

String Functions Practice:
Performing string manipulation operations
Applying functions like trim, concat, substring, replace, and case conversion 

String Data Number Practice:
Handling mixed data types and applying transformations
Combining numeric, string, and date operations with conditional logic 

3. Approach
   Worked on each module separately using problem statements and solutions

String Functions:
